# 🇺🇸 English

## Overview

Karuku is an AI-powered mathematics learning assistant designed to help students understand **how** and **why** a solution works.

Instead of only providing the final answer, Karuku analyzes handwritten solutions, detects mistakes, explains mathematical reasoning, and provides step-by-step feedback.

---

## Features

- ✍️ Handwritten equation recognition
- 📖 Step-by-step solution checking
- 🧠 Mathematical reasoning explanations
- ❌ Error detection
- 📈 Learning progress support
- 🌍 Multi-language ready

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- SQLite
- NextAuth.js
- Google Gemini API

---

## Vision

Mathematics is not about memorizing answers.

It is about understanding logical reasoning.

Karuku aims to become an intelligent learning companion that helps students build mathematical thinking.

# 🇯🇵 日本語

## 概要

Karuku は、途中式を重視した AI 数学学習支援アプリケーションです。

答えだけを返すのではなく、手書きの数式を認識し、途中式を解析し、誤りを検出し、どこでどのような考え方が必要だったのかを段階的に説明します。

---

## 主な機能

- ✍️ 手書き数式認識
- 📖 途中式の添削
- 🧠 数学的思考の説明
- ❌ 誤答箇所の検出
- 📈 学習履歴
- 🌍 多言語対応予定

---

## 技術スタック

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- SQLite
- NextAuth.js
- Google Gemini API

---

## ビジョン

数学は答えを覚える学問ではなく、論理を積み重ねる学問です。

Karuku は、一人ひとりの思考過程を支援する AI 学習環境を目指しています。
